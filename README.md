### How to Use a Theme

1. **Browse** through the theme gallery to find one you like
2. **Click** on any theme to see details and preview
3. Click the **"Apply"** button
4. AnymeX will open automatically and the theme will be added to your collection
5. **Switch** to your new theme in AnymeX

#### Switching Between Themes in AnymeX

1. Open AnymeX
2. Go to **Settings**
3. Navigate to **Player**
4. Select **Player Theme**
5. Choose any theme from your collection to apply it

That's it! Your new look is ready! üéâ

### Theme Structure

A theme is a JSON file that defines colors, typography, and visual effects for AnymeX.

```json
{
  "name": "My Awesome Theme",
  "description": "A beautiful dark theme for AnymeX",
  "creatorName": "Your Username",
  "category": "Dark",
  "colors": {
    "primary": "#6366f1",
    "secondary": "#8b5cf6",
    "background": "#171717",
    "foreground": "#fafafa",
    "card": "#262626",
    "border": "#404040",
    "muted": "#404040",
    "mutedForeground": "#a1a1aa",
    "accent": "#6366f1",
    "accentForeground": "#fafafa"
  },
  "typography": {
    "fontFamily": "Inter, sans-serif",
    "fontSize": {
      "small": "0.875rem",
      "medium": "1rem",
      "large": "1.25rem"
    }
  },
  "effects": {
    "blur": "8px",
    "shadow": "0 4px 6px -1px rgba(0, 0, 0, 0.1)",
    "borderRadius": "0.5rem"
  }
}
```

### Creating Your First Theme

#### Step 1: Choose Your Style

Decide on your theme's vibe:
- Dark, Light, or AMOLED
- Color palette (warm, cool, neon, pastel, etc.)
- Mood (minimal, vibrant, retro, modern)

#### Step 2: Design Your Color Palette

Pick colors that work well together:

**For Dark Themes:**
- Background: `#0a0a0a` to `#1a1a1a` (deep grays/blacks)
- Text: `#e5e5e5` to `#ffffff` (light grays/white)
- Accent: Any vibrant color that pops

**For Light Themes:**
- Background: `#fafafa` to `#ffffff` (near white)
- Text: `#171717` to `#262626` (dark grays)
- Accent: Any color that provides good contrast

**For AMOLED Themes:**
- Background: `#000000` (pure black)
- Text: `#ffffff` (pure white)
- Accent: High-contrast colors

#### Step 3: Create the JSON File

1. Create a new file named `my-theme.json`
2. Use the template above as a starting point
3. Customize the colors, fonts, and effects to match your vision
4. Test your theme in AnymeX
5. Iterate and refine

#### Step 4: Submit Your Theme

1. **Sign up** or **log in** to the AnymeX Themes platform
2. Go to the **Creator Dashboard**
3. Click **"Upload Theme"**
4. Fill in:
   - **Theme Name** - A catchy, descriptive name
   - **Description** - Tell users what makes your theme special
   - **Category** - Dark, Light, or AMOLED
5. **Upload** your JSON file (drag & drop or click to browse)
6. Click **"Upload Theme"** to submit!

üéâ Your theme will be **immediately visible** to all users in the gallery!

### Theme Submission Guidelines

#### ‚úÖ Do's

- Create original themes (don't copy others)
- Test your theme before submitting
- Use high-contrast colors for readability
- Write clear, engaging descriptions
- Choose appropriate categories
- Update your themes if users find issues

#### ‚ùå Don'ts

- Don't upload themes you didn't create
- Don't use copyrighted material
- Don't create themes that make text hard to read
- Don't include malicious code

### Managing Your Themes

In the Creator Dashboard, you can:
- **View** all your submitted themes
- **Edit** theme details (name, description, category)
- **Update** the JSON file
- **Delete** themes you no longer want to share
- **Track** stats (likes, views)

### Pro Tips for Great Themes

1. **Color Harmony** - Use color theory tools to create palettes that work well together
2. **Readability First** - Ensure text is always readable against backgrounds
3. **Consistent Spacing** - Use consistent padding and margins throughout
4. **Test in Different Environments** - Check how your theme looks in bright and dark rooms
5. **Get Feedback** - Share with friends before submitting
6. **Stay Updated** - Follow AnymeX updates to ensure compatibility
